# restaurant-management

# Front-end

---
#### Change Direcory:
```
cd client
```
#### Install Dependencies:
```
npm install
```

#### Start react app:
```
npm start
```


# Back-end

---
#### Change Direcory:
```
cd server
```
#### Install Dependencies:
```
npm install
```

#### Start server:
```
nodemon index.js
```
